# TaskVertex

A personal task manager to create, organize, and track your tasks — with priorities, due dates, and a clean dashboard.

Built with **React** (frontend) and **Spring Boot** (backend).

## Features

- JWT-based authentication (register & login)
- Create, edit, and delete tasks
- Set priority (High / Medium / Low) and due dates
- Filter tasks by status and priority
- Dashboard with task summary stats
- Overdue task highlighting

## Tech Stack

- **Frontend:** React, React Router, Axios, CSS
- **Backend:** Spring Boot, Spring Security, JWT
- **Database:** MySQL

## Getting Started

### Prerequisites
- Node.js 18+
- Java 17+
- MySQL running locally

### Frontend

```bash
cd frontend
npm install
npm start
```

Runs on `http://localhost:3000`
